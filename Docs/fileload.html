Cafe Sudoku

Loading:
Puzzles can be loaded from properly formatted
text files. A file is properly formatted if the
first 9 lines of the file are numbers separated
by spaces. For readibility, 0s may be represented
as "_".

A properly formatted line:
3 _ _ 2 _ 5 6 1 4


Cafe Sudoku will inform you if the file is not
properly formatted. If the file is formatted correctly,
it will automatically make it the current puzzle
being worked on.

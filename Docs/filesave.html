Cafe Sudoku

Saving:
Cafe Sudoku has the ability to
save the puzzle that you are working
on. Currently, Cafe Sudoku has the
ability to save the current final values
of the puzzle to a .txt file of your
choosing.

You may save your file choosing "Save"
from the File menu. You will be prompted
to name the puzzle if it has not been
named. If the puzzle has a name, clicking
Save will automatically update the file.


If you wish to save another copy of the
puzzle under a different name, you can
click "Save As...", which will let you
pick a name for the puzzle. As subsequent
saves will update this file.
